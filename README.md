Project Title: Bagel Bunch

Description: Project created in collaboration with Delfino, Gustavo, and Ryan. This project creates an interactive UI for commercial kitchenware specifically for a bagel oven.
Software Used: Qt Creator, Qt Designer
Hardware Used: Adafruit Trinkey SHT45
Languages Used: C++, Qt, QML

Installation Instructions:
1. Click the code
2. Grab the repository link
3. Clone the repository using Git/GitHub

Usage Examples:
Open the program and run using the Qt Creator IDE and have fun!
